Summary


Among many environmental issues, population growth is of paramount importance because this factor has a great physical impact on its regional and global environment. This website aims to investigate this major factor and present the result by statistical visualisation.

Data


The analysis is based on the 10 most populated Countries and cities for an 8-year period, and predicts their populations in 2030 and 2050. In addition, the growth rate and population ratio are studied in this project.

Generally, in this analysis an attempt has been made to answer the following questions:

  1. What are the 10 countries with the largest population?
  2. What is the percentage of the top 10 countries compared to the rest of the world?
  3. Are the top 10 cities located in the top 10 countries?
  4. Change in population from 2019 to 2020?

Dataset


For this purpose we scraped our dataset from https://worldpopulationreview.com

Process


We tried to follow all the requirements included in the instructions.

First, we scraped data from a valid website. Then, cleaned the dataset (ELT) and created a flask python for other users's convenience. Using JavaScript for creating the map and plots and finally, presented the data on dashboard by using html and css.

Tools

The following tools have been used in this project:

  • Scrape data/ELT: Python, Pandas
  • Database/ELT: MongoDB
  • API: Flask
  • Visualisation: Javascript (D3, Plotly, Leaflet, jSuites)
  • HTML/CSS: BootStrap
