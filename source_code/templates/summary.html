Summary


The current US Census Bureau world population estimate in June 2019 shows that the current global population is 7,577,130,400 people on earth, which far exceeds the world population of 7.2 billion from 2015. Our dataset estimate based on UN data shows the world's population surpassing 7.7 billion.

Many other nations have populations of at least one million, while there are also countries that have just thousands. The smallest population in the world can be found in Vatican City, where only 801 people reside.

Although the number presented in our project seems very precise, it is important to remember that it is just an estimate. It simply isn't possible to be sure exactly how many people there are on Earth at any one time.

Data


The analysis is based on the 10 most populated Countries and cities for an 8-year period, and predicts their populations in 2030 and 2050. In addition, the growth rate and population ratio are studied in this project.

Generally, in this analysis an attempt has been made to answer the following questions:

  1. What are the 10 countries with the largest population in 2020?
  2. What is the percentage of the top 10 countries compared to the rest of the world in 2020?
  3. What are the top 10 cities with the largest population? in 2020?
  4. Is the largest populated city located in the largest populated country in the world?
  5. Predicted population per country from 2020 to 2050 (eg. Australia)?

Process


As you can see in our flowchart, after performing the ETL and loading the Data into a MongoDB database, we use Flask to create the API endpoints (point to Endpoints) for accessing the data in JavaScript. We used D3 library for reading the data from our API endpoints which are in json format, and our Web Application has these (point to Web App) web pages to display our plots and findings.

Tools

The following tools have been used in this project:

  • Scrape data/ELT: Python, Pandas
